Implementation and Visualization of Numerical Algorithms

Numerical Analysis is the area of Mathematics that deals with creating, analyzing and studying Mathematical problems with the help of computers. It has a wide variety of real-world applications throughout the natural sciences, engineering, medicine, and even in business.

The aim of this website is to provide a simple and easy to use interface along with useful visualiazations to some common Numerical Analysis algorithms that are accessible to everyone without skimming over the important technical details. The motivation being that Numerical Analysis is a subject at the intersection of Mathematics and Computer Science and its beauty should not be lost in hand-calculations.

The website is divided into 6 main modules which can be accessed using the navigation bar above. Each module is independant and self-contained. Each module solves a different problem and a simple description (in layman terms) is provided at the begining of each module as well.

All the algorithms have been written using Python and this website is hosted on Heroku
