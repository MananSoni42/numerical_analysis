Implementation and Visualization of Numerical Algorithms

“ The purpose of computing is insight, not numbers “
- Richard Hamming

Numerical Analysis is an area of Mathematics and Computer Science that is almost as old as computers themselves! It is concerned with all aspects of the numerical solution of a problem, from the theoretical development and understanding of numerical methods to their practical implementation as reliable and efficient computer programs.
It naturally finds application in all fields of engineering and the physical sciences, but in the 21st century also the life sciences, social sciences, medicine, business and even the arts have adopted elements of scientific computations.

The aim of this website is to provide a simple and easy to use interface that offers useful visualiazations to some common problems in Numerical Analysis. These problems are (generally) taught in an undergraduate course in Numerical Analysis.

The website is divided into 6 main modules which can be accessed using the navigation bar above. Each module is independant and self-contained. Each module solves a different problem and a simple description (in layman terms) is provided at the begining of each module as well.

All the algorithms have been written using Python and this website is hosted on Heroku
The code is completely open source and licensed under the MIT license
